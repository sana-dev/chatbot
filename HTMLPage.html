How can I ingest data using TimeXtender?

X-Pilot

To ingest data using TimeXtender, you can follow these steps:
  1. First, open the TimeXtender project that you want to ingest data into.
  2. Next, navigate to the Data Warehouse tab in the project.
  3. Click on the "Data Sources" button to add a new data source.
  4. Select the type of data source you want to ingest data from, such as a database or a file.
  5. Provide the necessary connection details for the data source, such as the server name, database name, and credentials.
  6. Once the data source is added, you can configure the tables and columns that you want to ingest data from.
  7. Map the columns from the data source to the corresponding columns in your data warehouse.
  8. Set up any necessary transformations or business rules for the data ingestion process.
  9. Finally, run the data ingestion process to import the data from the source into your data warehouse.
By following these steps, you can successfully ingest data using TimeXtender.
